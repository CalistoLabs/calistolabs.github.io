SANYO North America

Corporate/E-Commerce Site

‹ ›
Tags:
asp.net, sql-server, paypal-api, c#, art direction, visual strategy, ui & ux design

The SANYO project involved a complete redesign of the site including restructuring the navigation to be more intuitive for users and the development of a comprehensive content management system to help administrators easily maintain, contribute and moderate content for all the three regional sites of SANYO North America.

It also included development of the SANYO Store, a complete e-commerce solution with PayPal PayFlow Pro API as the payment processing back-end featuring order notification, automatic inventory updates, real-time shipping rate calculation via UPS API, discount vendor/employee/sale pricing, promotions, advance taxing and shipping rules among other things all controlled using the CMS.

Created design comps, worked on entire flow, and came up with design solutions to present a massive amount of content. Also created animations, CSS framework and an HTML style guide.
