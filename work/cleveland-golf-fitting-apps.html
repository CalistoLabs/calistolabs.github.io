Cleveland Golf Fitting Apps

Interactive Application

‹ ›
Tags:
laravel, php, mysql, jquery, ui & ux design

Cleveland Golf Wedge, Iron and Driver Fitting Applications were designed and built to guide users and retailers through a fitting process through which they are given various options to customize their gear.

Developed the backend to handle user authentication, store order history, and to show sales reports via an admin interface using Laravel. Also developed the frontend using jQuery.

Designed a flexible layout for each screen to work on both desktop and mobile. Created custom responsive grid and worked on HTML & CSS.
